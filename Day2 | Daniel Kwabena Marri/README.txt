Module 2 -- Analyze Covid-19 RNA Seq Data ( Finding differentially expressed genes using DESeq2).

This module is intended to give an insight to analyzing Covid-19 RNA-Seq dataset. Finding differentially expressed genes
the RNA-Seq data using the library DESeq2. Attendees will gain experience in using R-Studio to analyse RNA-Seq dataset.

Prerequisites Installation : 
All of the work was done in R-Studio which can be installed after installing R 4.0 or above.

R-installation link:

For Windows:
https://cran.r-project.org/bin/windows/base/

For Mac:
https://cran.r-project.org/bin/macosx/

R-Studio installation link:
https://www.rstudio.com/products/rstudio/

R - Library packages to install before the session:

1. DESeq2
2. RColorBrewer
3. ggplot2
4. dplyr
